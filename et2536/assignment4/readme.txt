Prerequisites:
              Perl module Net::SNMP & DBI modules should be installed.
              Install Linux LAMP Stack (Includes Php , Mysql, Apache ) in one package using command "sudo apt-get install lamp-server^"

Procedure:
          1)Replace db.conf file with your own db file or change the credentials accordingly.
          2)Use "cd" command to change directory to assignment1 folder.
          3)Run command "sudo perl backend.pl", leave terminal or send to backgroup process.
          4)Using browser open appropriate URL
		  Please see: The duplicate entries are prevented, so tables must be dropped when ever program is interrupted
